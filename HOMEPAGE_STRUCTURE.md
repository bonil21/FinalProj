## 🎨 Color Scheme Reference

### Primary Colors
- **Deep Green**: #2F5133 (main text, headers)
- **Medium Green**: #3B6B4A (buttons, secondary)
- **Bright Green**: #6BB97F (accents, hovers)

### Accent Colors
- **Warm Orange**: #FFA62B (CTAs, highlights)
- **Soft Orange**: #FFB84D (hover state)

### Neutral Colors
- **Warm Beige**: #F6E6D7 (background)
- **Light Green**: #E7F0DC (card backgrounds)
- **Dark Text**: #1f2f24, #2F5133
- **Gray Text**: #5f6b74, #6b7280
- **Border**: #E4CFBF

---

## 📏 Responsive Grid System

### Desktop (1200px+)
- **Features**: 3 columns
- **Categories**: 4-5 columns
- **Products**: 4 columns
- **Blog**: 3 columns
- **Deals**: 2 columns

### Tablet (768px-1199px)
- **Features**: 1 column
- **Categories**: 2-3 columns
- **Products**: 3 columns
- **Blog**: 2 columns
- **Deals**: 2 columns

### Mobile (640px-767px)
- **All Grids**: 2 columns (except single column for products)
- **Blog**: 1 column
- **Deals**: 1 column

### Small Mobile (<640px)
- **All Grids**: 1 column
- **Full width stacking**

## 🎯 Key Components

### Buttons
- **Primary**: Green gradient with shadow
- **Secondary**: Orange/warm colors
- **Sizes**: Default (12px padding), Large (16px padding)
- **Hover**: Lift effect, color shift, shadow increase

### Cards
- **Background**: White or light green
- **Shadows**: Subtle, increasing on hover
- **Borders**: 1px solid #E4CFBF
- **Radius**: 16px
- **Hover**: translateY(-5px)

### Forms
- **Input**: Rounded (25px), light background
- **Focus**: #6BB97F border, shadow
- **Button**: Green gradient

### Images
- **Ratio**: 1:1 for products, 16:9 for blog
- **Hover**: Scale 1.05 transform
- **Radius**: 10-12px

---

## 📱 Mobile First Approach

The design uses mobile-first CSS with media queries for larger screens:

```css
/* Base (mobile) styles */
.element { /* optimized for small screens */ }

/* Tablet and up */
@media (min-width: 768px) {
  .element { /* tablet adjustments */ }
}

/* Desktop and up */
@media (min-width: 1200px) {
  .element { /* desktop optimizations */ }
}
```

---

## ✨ Interactive Elements

### Hover Effects
- **Cards**: Lift (Y -5px) + shadow increase
- **Buttons**: Lift (Y -2px) + color shift + shadow increase
- **Links**: Color change (#2F5133 → #6BB97F)
- **Images**: Scale (1.0 → 1.05)

### Transitions
- **Duration**: 0.3s ease
- **Properties**: all, transform, box-shadow, color

### Visual Feedback
- **Focus States**: Border color change, shadow
- **Active States**: Darker background
- **Disabled States**: Reduced opacity (not used currently)
